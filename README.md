# 💳 FlexiPay Gateway for WooCommerce

FlexiPay Gateway for WooCommerce is a powerful plugin that allows you to create and manage upto 10 custom offline payment methods with ease. Add payment instructions, QR codes, and handle manual payments efficiently.

---

## 🚀 Features

* ✅ Create up to **10 custom payment gateways**
* 🧾 Add **custom instructions** for each gateway
* 📷 Upload and display **QR codes**
* 📧 Send payment details via **email notifications**
* ⚙️ Enable/disable gateways from WooCommerce settings
* 🎯 Clean and simple admin interface
* 🔒 Built with WordPress security best practices

---

## 📦 Requirements

* WordPress 5.0 or higher
* WooCommerce (installed & activated)
* PHP 7.2 or higher

---

## 🛠 Installation

1. Clone or download this repository:

   ```bash
   git clone https://github.com/your-username/flexipay-gateway.git
   ```

2. Upload the plugin to:

   ```
   /wp-content/plugins/flexipay-gateway
   ```

3. Activate the plugin from **WordPress Admin > Plugins**

4. Go to:

   ```
   Settings > FlexiPay Gateway
   ```

5. Create your custom payment gateways

6. Enable them from:

   ```
   WooCommerce > Settings > Payments
   ```

---

## ⚙️ Usage

1. Add a new gateway from plugin settings
2. Configure:

   * Gateway name
   * Instructions
   * QR code image
3. Enable the gateway in WooCommerce
4. Customers can now select it at checkout

---

## 📧 Email Integration

* Payment instructions are sent in **On-Hold order emails**
* Option to include:

  * Selected gateway details
  * All available payment methods
